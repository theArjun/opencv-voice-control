## [Unreleased]

### Added
- Gesture-controlled macOS master volume: pinch thumb and index finger to set
  the system volume in real time.
- Hand tracking via MediaPipe's `HandLandmarker` (Tasks API).
- On-screen HUD with hand skeleton, volume bar, percentage, and FPS.
- Scale-invariant pinch mapping (normalized by hand size), EMA smoothing, and
  throttled volume updates.
- Project documentation: README, CONTRIBUTING, CODE_OF_CONDUCT, LICENSE (MIT),
  and GitHub issue/PR templates.
